# SubTrack – Smart Subscription Tracker

SubTrack is an Android application designed to help users track and manage their subscription services in a simple and privacy-friendly way. The app allows users to manually add and monitor subscriptions without linking their bank accounts.

Many users forget about recurring payments from services like streaming platforms, software tools, and memberships. SubTrack helps users stay aware of their expenses by providing reminders, spending insights, and cancellation guidance.

---

## Features

- User authentication with login and Google Sign-In  
- Add, edit, and manage subscriptions  
- Track total monthly subscription spending  
- View upcoming renewal dates  
- Smart reminder notifications for renewals  
- Tutorial videos explaining how to cancel popular subscriptions  
- Custom alert sounds for different reminder levels  
- Clean and responsive mobile UI

---

## Technologies Used

- **Kotlin**
- **Android Studio**
- **Material Design Components**
- **ConstraintLayout**
- **VideoView** for tutorial playback
- **MediaPlayer** for notification sound previews
- **Google Play Services Location API**

---

## APK Installation

1. Download the **SubTrack.apk** from this repository.
2. Transfer the APK file to your Android device.
3. Enable **Install from Unknown Sources** in device settings.
4. Tap the APK file and install the application.

---

## Requirements

- Android **API 24 (Android 7.0)** or higher
- Internet connection (for streaming tutorial videos)
